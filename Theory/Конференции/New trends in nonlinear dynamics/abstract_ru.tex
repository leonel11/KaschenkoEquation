Рассмотрим уравнение параболического типа с линейным отклонением в краевом условии:

\begin{equation}\label{ivanovsky-eq1}
	\dot u = \beta {u''} - \gamma u,	
\end{equation}
\begin{equation}\label{ivanovsky-eq1:1}  
u'\mid_{x=0} \, = 0,  \quad  u'\mid_{x=1} \, = \alpha\,u\mid_{x=x_0}, 
\end{equation}
где скалярная функция $ u = u(x, t) $ дифференцируема по $t$ и дважды дифференцируема по $x$, параметры $ \alpha, \gamma \in \mathbb{R}, \; \beta > 0, \; x_0 \in [0, 1] $.

С помощью замены вида $ u(x, t) = w(x)\,\mbox{exp}\left( \lambda - \frac{\gamma}{\beta}t \right) $ 
краевая задача \eqref{ivanovsky-eq1}, \eqref{ivanovsky-eq1:1} сводится к задаче на собственные значения вида

\begin{equation}\label{ivanovsky-eq2}
	w'' - \lambda w = 0,
\end{equation}
\begin{equation}\label{ivanovsky-eq2:1}  
 w'(0) = 0,  \quad  w'(1) = \alpha\,w(0).
 \end{equation}
Решение уравнения \eqref{ivanovsky-eq2}, удовлетворяющее первому из краевых условий 
\eqref{ivanovsky-eq2:1}, записывается следующим образом:
\begin{equation}\label{ivanovsky-eq5}  
w(x) = c\,{\rm ch\,}(\mu\, x), 
 \end{equation}
где $ c $ -- константа, а $ \mu = \sqrt{\lambda} $. Тем самым, для нахождения собственных значений краевой задачи 
 \eqref{ivanovsky-eq2},  \eqref{ivanovsky-eq2:1} необходимо функцию \eqref{ivanovsky-eq5}  подставить во второе краевое условие  \eqref{ivanovsky-eq2:1}. 

Для выяснения свойств устойчивости нулевого решения краевой задачи \eqref{ivanovsky-eq1}, \eqref{ivanovsky-eq1:1} необходимо найти собственное число $\lambda$, вещественная часть которого максимальна. В этом случае представляет интерес ситуация, когда все собственные числа задачи лежат в левой комплексной полуплоскости и одна пара из них находится на мнимой оси. 
Для решения этой задачи найдем такое значения параметра $ \alpha_{cr} $, для которого выполняется условие $ \mbox{Re}(\lambda_*)  = \gamma $, тогда если удается показать, что все остальные собственные числа лежат в левой комплексной полуплоскости, то можно говорить о колебательной потере устойчивости нулевого состояния равновесия задачи \eqref{ivanovsky-eq1},  \eqref{ivanovsky-eq1:1}. Поскольку найти нужные значения $ \alpha_{cr} $ аналитически довольно затруднительно, исследование осуществлялось с помощью специально разработанного приложения. Все трудоемкие расчеты в программе выполняются параллельно, на независимых потоках центрального процессора.

В результате численного эксперимента были получены зависимости значений $ \alpha_{cr} $ от параметров $ \gamma $ и $ \lambda $, для которых нулевое решение уравнения \eqref{ivanovsky-eq1},  \eqref{ivanovsky-eq1:1} колебательно теряет устойчивость.

 Работа выполнена при поддержке гранта Российского научного фонда (\textnumero~14-21-00158).
