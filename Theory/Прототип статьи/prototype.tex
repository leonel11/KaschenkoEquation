\section*{Аннотация}

В статье рассматривается краевая задача с линейным отклонением в краевом условии. Для нее изучаются вопросы колебательной и дивергентной потери устойчивости нулевого состояния равновесия, а также какие при этом решения от него ответвляются. Для поставленной задачи в комплексе применяются аналитические и численные методы для решения. При численном исследовании динамической системы, представляющей собой цепочку из связанных осцилляторов, моделирующую изначальную краевую задачу. При исследовании особое внимание уделяется значениям начальных параметров, при которых нулевое решение краевой задачи меняет свою устойчивость.

\section{Постановка задачи}

Рассмотрим краевую задачу c линейным отклонением в одном из краевых условий
\begin{equation}\label{ivanovsky-eq1}
	\dot u = u'' + \gamma u - u^3,	
\end{equation}
\vspace{-1cm}
\begin{equation}\label{ivanovsky-eq2}	
	u'(0, t) \, = 0, \qquad u'(1, t) \, = \alpha\,u(0, t),
\end{equation}
где параметры $ \alpha, \gamma \in \mathbb{R} $. Краевая задача \eqref{ivanovsky-eq1}, \eqref{ivanovsky-eq2} очевидным образом имеет нулевое решение. В зависимости от значений параметров, это решение может быть устойчивым или неустойчивым. Представляет интерес определить условие устойчивости этого состояния равновесия и выяснить какие решения от него ответвляются при потере устойчивости. В данном случае основными способами ее потери являются дивергентный, когда в спектре устойчивости состояния равновесия появляется нулевое значение, и колебательный, соответствующий случаю выхода пары собственных значений  на мнимую ось.

\section{Линейный анализ}

Как и в статье [1], рассмотрим линеаризованную в нуле краевую задачу \eqref{ivanovsky-eq1}, \eqref{ivanovsky-eq2}:
\begin{equation}\label{ivanovsky-eq3}
	\dot u = u'' + \gamma u,	
\end{equation}
\vspace{-0.8cm}
\begin{equation}\label{ivanovsky-eq4}	
	u'(0, t) \, = 0, \qquad u'(1, t) \, = \alpha\,u(0, t),
\end{equation}
Для выяснения устойчивости нулевого решения выполним стандартную эйлерову замену вида $ u(x, t) = e^{\lambda t} \, v(x) $. Тогда для функции $ v(x) $ получим следующую задачу на собственные значения:
\begin{equation}\label{ivanovsky-eq5}
	v'' + (\gamma - \lambda)v = 0,	
\end{equation}
\vspace{-0.8cm}
\begin{equation}\label{ivanovsky-eq6}	
	v'(0) \, = 0, \qquad v'(1) \, = \alpha\,v(0),
\end{equation}
Решая дифференциальное уравнение \eqref{ivanovsky-eq5} и применяя первое из краевых условий \eqref{ivanovsky-eq6}, получаем, что 
\begin{equation}\label{ivanovsky-eq7}
	v(x) = c \, \mbox{ch} x,
\end{equation}
где $ c \in \mathbb{R} $ --- константа. Применяя к формуле \eqref{ivanovsky-eq7} второе краевое условие из \eqref{ivanovsky-eq6}, имеем следующее выражение
\begin{equation}\label{ivanovsky-eq8}
	\sqrt{-\gamma + \lambda} \; \mbox{sh} \sqrt{-\gamma + \lambda} \, = \, \alpha ,
\end{equation}
которое представляет собой характеристическое уравнение краевой задачи \eqref{ivanovsky-eq3}, \eqref{ivanovsky-eq4}. Здесь выяснение устойчивости нулевого состояния равновесия сталкивается со следующими трудностями: мы можем найти значения, при которых корни характеристического уравнения \eqref{ivanovsky-eq8} пересекают мнимую ось. Однако в таком случае не удается доказать, что все остальные корни будут лежать слева от мнимой оси. В связи с этим будем в комплексе применять аналитические и численные методы для решения данной задачи.

Выясним сначала важный вопрос, при каких критических значениях $\alpha_{cr}(\gamma)$ корни характеристического уравнения \eqref{ivanovsky-eq8} выходят на мнимую ось. Выделим здесь два случая. В первом случае будем считать, что собственное число $ \lambda $ обращается в $ 0 $, т.е. происходит так называемая дивергентная потеря устойчивости. Подстановка $ \lambda = 0 $ в характеристическое уравнение \eqref{ivanovsky-eq8} приводит к аналитической зависимости следующего вида
\begin{equation}\label{ivanovsky-eq9}
	\alpha_{cr}(\gamma) = \sqrt{-\gamma} \; \mbox{sh} \sqrt{-\gamma}
\end{equation}
На рис. 1 эта кривая изображена синим цветом и обозначена как $ \alpha_d $.

Теперь переходим к построению критической зависимости $ \alpha_{cr}(\gamma) $ в случае колебательной потери устойчивости, т.е. считаем, что в уравнении \eqref{ivanovsky-eq8} $ \lambda = i \omega $, где $ \omega \in \mathbb{R} $. Тогда для критических значений $ \alpha $ получаем следующее выражение:
\begin{equation}\label{ivanovsky-eq10}
	\alpha_{cr}(\gamma) = \sqrt{-\gamma + i \omega} \; \mbox{sh} \sqrt{-\gamma + i \omega} \,
\end{equation}
Кривую, соответствующую этому уравнению, можно получить лишь численно. На рис. 1 она показана красным цветом и обозначена как $ \alpha_c $.

Существует также иной способ получения кривой $ \alpha_c $. Он связан с численным решением краевой задачи \eqref{ivanovsky-eq3}, \eqref{ivanovsky-eq4} и поиском таких критических значений $ \alpha $, при которых нулевое решение теряет устойчивость. Для этого рассмотрим цепочку из $ n $ связанных осцилляторов, численно моделирующую краевую задачу \eqref{ivanovsky-eq1}, \eqref{ivanovsky-eq2}:
\begin{equation}\label{ivanovsky-eq11} 
	\dot{u}_j =  n^2(u_{j+1} - 2u_j + u_{j-1}) + \gamma u_j - u_j^3, \quad j = \overline{1, n}. 
\end{equation}
При этом краевые условия заменяются на
\begin{gather}\label{ivanovsky-eq12}	
	u_0 = u_1, \quad u_{n+1} = u_n + \frac{\alpha_c}{n}\:u_1.
\end{gather}
Численный способ получения кривой $ \alpha_c $ состоит в решении линеаризованной системы дифференциальных уравнений \eqref{ivanovsky-eq11} с краевыми условиями (12), для которой значение $ \alpha $ при заданном параметре $ \gamma $ выбирается из условий устойчивости или неустойчивости нулевого состояния равновесия. 

Данное исследование осуществлялось с помощью специально разработанного приложения. При численном моделировании уравнения \eqref{ivanovsky-eq11}, величина $ n $ считалась равной 50. Графическая визуализация численного эксперимента приведена на рис. 2.

Эти кривые являются важнейшими элементами для построения областей значений параметров $ \alpha $ и $ \gamma $, определяющих устойчивость нулевого состояния равновесия краевой задачи \eqref{ivanovsky-eq3}, \eqref{ivanovsky-eq4}.

\section{Локальный анализ краевой задачи (1), (2)}

В этом разделе методами возмущений построим режим, ответвляющийся от нулевого состояния равновесия задачи \eqref{ivanovsky-eq1}, \eqref{ivanovsky-eq2}.

\subsection{Случай дивергентной потери устойчивости}

Для изучения фазового портрета краевой задачи \eqref{ivanovsky-eq1}, \eqref{ivanovsky-eq2} около нулевого состояния равновесия используется нормальная форма, которая получается в результате разложения решения по степеням малого параметра
\begin{equation}\label{ivanovsky-eq13}
	u = \sqrt{\varepsilon}u_0 + \varepsilon u_1 + \varepsilon^{\frac{3}{2}} u_2 + O(\varepsilon^2).
\end{equation}
Малый параметр $ \varepsilon=\alpha-\alpha_{d} $, где $ \alpha_{d} $ вычисляется по формуле \eqref{ivanovsky-eq9}, косвенно характеризует собой отклонение от положения равновесия. Для нормальный формы \eqref{ivanovsky-eq13} функция $ u_0 $, в случае дивергентной потери устойчивости принимает вид
\begin{equation}\label{ivanovsky-eq14}
	u_0 = \rho(s) v(x),
\end{equation}
где параметр $ s = \varepsilon t $ --- медленное время. Подстановка в уравнение \eqref{ivanovsky-eq1} разложения \eqref{ivanovsky-eq13} с учетом \eqref{ivanovsky-eq14} приводит к системе последовательно разрешимых краевых задач следующего вида
$$	v'' + \gamma v = 0, $$
\vspace{-0.8cm}
\begin{equation}\label{ivanovsky-eq15}
	v'(0) = 0, \quad v'(1) = \alpha_{d}\,v(x_0),
\end{equation}

\begin{equation}\label{ivanovsky-eq16}
	\frac{\partial u_0}{\partial s} + \dot{u}_2 = u''_2 + \gamma u_2 - u_0^3,
\end{equation}
\vspace{-0.7cm}
\begin{equation}\label{ivanovsky-eq17}
u_2'(0, t) = 0, \quad u_2'(1, t) = \alpha_d\,u_2(x_0, t) + u_0(x_0, t).
\end{equation}
Граничные условия \eqref{ivanovsky-eq15} и \eqref{ivanovsky-eq17} формируются путем подстановки формулы \eqref{ivanovsky-eq13} с учетом \eqref{ivanovsky-eq14} в краевые условия \eqref{ivanovsky-eq2}.

Решая краевую задачу\eqref{ivanovsky-eq16}, \eqref{ivanovsky-eq17} можно получить уравнение на ляпуновские величины следующего вида
\begin{equation}\label{ivanovsky-eq18}
	\rho' = \phi_0 \rho + d_0 \rho^3,
\end{equation}
$$ \phi_0 = \frac{ 2 \sqrt{-\gamma} }{ \sqrt{-\gamma} \, \mbox{ch} \sqrt{-\gamma} +\mbox{sh} \sqrt{-\gamma} }, $$
$$ d_0 = \frac{ -3 \gamma \mbox{sh} 3 \sqrt{-\gamma} - 12 \mbox{sh} \sqrt{-\gamma} - 12 \sqrt{-\gamma} \mbox{ch} \sqrt{-\gamma} - a_d \sqrt{-\gamma} }{ 16 ( \sqrt{-\gamma} \, \mbox{ch} \sqrt{-\gamma} + \mbox{sh} \sqrt{-\gamma} ) }. $$
Исходя из полученных результатов, были выведены следующие теоремы.
\newtheorem{theorem}{Теорема} 
\begin{theorem}
Если в уравнении \eqref{ivanovsky-eq18} $ \phi_0>0 $, а  $ d_0<0 $, то  $ \exists \, \varepsilon_0>0: \, \forall \varepsilon \in (0,\varepsilon_0] $ в фазовом пространстве краевой задачи \eqref{ivanovsky-eq1}, \eqref{ivanovsky-eq2} при определенных значениях параметра $ \gamma $ нулевое состояние равновесия теряет свою устойчивость.
\end{theorem}

\begin{theorem}
Пусть $ \gamma < \gamma_l, $ где $ \gamma_l $ -- корень уравнения
$$ -3 \gamma \sin 3 \sqrt{\gamma} - 12 \sin \sqrt{\gamma} - 12 \sqrt{\gamma} \cos \sqrt{\gamma} + \gamma \sin \sqrt{\gamma} = 0. $$
Тогда $ \exists \, \varepsilon_0 > 0: \, \forall \varepsilon \in (0, \varepsilon_0] $ краевая задача \eqref{ivanovsky-eq1}, \eqref{ivanovsky-eq2} имеет 2 локально устойчивых пространственно-неоднородных состояния равновесия. 
\end{theorem} 
\newpage
\begin{theorem}
Пусть $ \gamma_l < \gamma < \gamma_{*}, $ где $ \gamma_{*} $ -- корень уравнения $ \sqrt{\gamma} = -\tg \sqrt{\gamma} $. Тогда при $ \alpha > \alpha_d $ нулевое решение краевой задачи \eqref{ivanovsky-eq1}, \eqref{ivanovsky-eq2} жестко теряет устойчивость при достаточно малой разности $ \alpha_d - \alpha $, а к нулевому состоянию равновесия приближаются 2 неустойчивых состояния равновесия. 
\end{theorem}

\subsection{Случай колебательной потери устойчивости}

В случае колебательной потери устойчивости будем действовать аналогично. Используя нормальную форму \eqref{ivanovsky-eq13}, для краевой задачи \eqref{ivanovsky-eq1}, \eqref{ivanovsky-eq2} отыщем решения по степеням корня малого параметра $ \varepsilon = \alpha-\alpha_c $, где $ \alpha_c $ вычисляется по формуле \eqref{ivanovsky-eq10}. Считая, что $ \lambda = i \omega $ для $ \omega \in \mathbb{R} $, функция $ u_0 $ будет принимать вид
\begin{equation}\label{ivanovsky-eq19}
	u_0 = z(s) e^{i \omega t} v(x),
\end{equation}
Подстановка в уравнение \eqref{ivanovsky-eq1} разложения \eqref{ivanovsky-eq13} с учетом \eqref{ivanovsky-eq19} приводит к системе последовательно разрешимых краевых задач следующего вида 
$$	v'' + \gamma v = 0 $$
\vspace{-1cm}
\begin{equation}\label{ivanovsky-eq20}
	v'(0) = 0, \quad v'(1) = \alpha_c\,v(x_0),
\end{equation}

\begin{equation}\label{ivanovsky-eq21}
	\frac{\partial u_0}{\partial s} + \dot{u}_2 = u''_2 + \gamma u_2 - u_0^3
\end{equation}
\vspace{-0.7cm}
\begin{equation}\label{ivanovsky-eq22}
u_2'(0, t) = 0, \quad u_2'(1, t) = \alpha_c\,u_2(x_0, t) + u_0(x_0, t).
\end{equation}
Граничные условия \eqref{ivanovsky-eq20} и \eqref{ivanovsky-eq22} для каждой краевой задачи формируются путем подстановки формулы \eqref{ivanovsky-eq13} с учетом \eqref{ivanovsky-eq19} в краевые условия \eqref{ivanovsky-eq2}.

Решая краевую задачу \eqref{ivanovsky-eq21}, \eqref{ivanovsky-eq22} можно получить уравнение на ляпуновские величины следующего вида
\begin{equation}\label{ivanovsky-eq23}
	z' = \phi z + d z |z|^2.
\end{equation}
$$ \phi = \frac{ 2 \mu }{ \mu \, \mbox{ch} \mu +\mbox{sh} \mu }, $$
$$ d = \frac{ 3 \mu ( G(\mu + 2\,\mbox{Re}\mu) + G(\mu + 2i\,\mbox{Im}\mu) + 2G(\overline{\mu}) ) }{ 2 ( \mu \, \mbox{ch} \mu +\mbox{sh} \mu ) }, $$
где $ \mu = \sqrt{-\gamma + i\omega} $, а $ G(y) $ -- комплекснозначная функция, вычисляемая по формуле
$$ G(y) = \frac{ \alpha_c - y\,\mbox{sh} y }{ y^2 + \gamma - i\omega }. $$

Для исследования динамики фазового портрета в данном случае, как и в случае дивергентной потери устойчивости достаточно изучить зависимость значений $ \phi_0 = \mbox{Re}\,\phi $ и $ d_0 = \mbox{Re}\,d $ уравнения \eqref{ivanovsky-eq23} от параметра $ \gamma $.
