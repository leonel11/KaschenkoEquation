\section*{Постановка задачи}

Рассмотрим краевую задачу параболического типа с интегральной внутренней связью во втором краевом условии:
\begin{equation}\label{system5}
\dot{u} = u'' + \gamma u - u^3, 
\end{equation}
\begin{equation}\label{system6}
u'(0, t) \, = 0, \qquad u'(1, t) \, = \alpha\int\limits_{0}^{1} u(t, y) dy.
\end{equation}
Здесь $ u(x, t)$ --- гладкая функция для времени $t \geqslant 0$ и пространственной координаты $x \in [0,1]$, параметры $ \alpha $ и $ \gamma$ --- действительные числа, а величина $ x_0 \in [0, 1) $ позволяет определить дополнительную связь в краевом условии.

Наряду с краевой задачей \eqref{system5}, \eqref{system6} введем в рассмотрение соответствующую ей систему дифференциальных уравнений с диффузионным взаимодействием:
\begin{equation}\label{original_system}
\dot{u}_j =  N^2(u_{j+1} - 2u_j + u_{j-1}) + \gamma u_j - u_j^3, \qquad j = \overline{1, N}, 
\end{equation}
\begin{equation}\label{original_conditions}
u_0 = u_1, \; u_{N+1} = u_N + \dfrac{\alpha}{N^2} \sum_{k=1}^{n} \:u_k, \qquad 1 \leqslant k < N,
\end{equation}
где $ u_j = u_j(t) $ гладкие функции при $ t \geqslant 0 $, величина $ N \gg 1 $, а индекс $ k \in \mathbb{N} $ вместо параметра $ x_0 $ позволяет определить дополнительную внутреннюю связь между элементами $ u_N $ и $ u_k $. Такие системы часто изучаются как модели связанных осцилляторов (см. например \cite{GKR_khaos}, \cite{GKR_relax}), в которых взаимодействие происходит не только между соседними элементами, но и с каким-нибудь внутренними элементами цепочки.

Система \eqref{original_system}, \eqref{original_conditions}, как и краевая задача \eqref{system5}, \eqref{system6}, очевидным образом имеет однородное нулевое решение $ u_j(t) \equiv 0 $. Представляет интерес вопрос устойчивости этого решения при переходе через мнимую ось нулевого собственного числа. Задача исследования состояла в изучении характера потери устойчивости нулевого решения системы \eqref{original_system}, \eqref{original_conditions}, и, как следствие, краевой задачи \eqref{system5}, \eqref{system6}, т.е. в поиске критических значений параметров $\alpha, \, \gamma$ и получении асимптотических формул для режимов, ответвляющихся от нулевого решения.

\section{Спектральные свойства линеаризованной системы}

Рассмотрим линеаризованную в нуле систему дифференциальных уравнений \eqref{original_system}, \eqref{original_conditions}:
\begin{equation}\label{linearized_system}
\dot{u}_j =  N^2(u_{j+1} - 2u_j + u_{j-1}) + \gamma u_j, \qquad j = \overline{1, N}, 
\end{equation}
\begin{equation}\label{linearized_conditions}
u_0 = u_1, \; u_{N+1} = u_N + \dfrac{\alpha}{N^2} \sum_{k=1}^{n} \:u_k, \qquad 1 \leqslant k < N.
\end{equation}
Для определения условий устойчивости нулевого решения выполним стандартную эйлерову замену вида
\begin{equation}\label{euler_replace}
u_j(t) = e^{\lambda t}\ch \delta x_j 
\end{equation}
где $ x_j = -\dfrac{1}{2N} + \dfrac{j}{N} $, $ \lambda $ –- собственное значение матрицы линеаризованной системы, а коэффициент $ \delta $ определяет собственный вектор соответствующего собственного числа матрицы системы \eqref{linearized_system} с условиями \eqref{linearized_conditions}. При ее подстановке в первое уравнение, для $ j=1 $
$$ \dfrac{\lambda - \gamma}{4 N^2} = \sh^2 \dfrac{\delta}{2 N} $$
и решении этого уравнения, можно получить следующую формулу для коэффициента $ \delta $:
$$ \delta = 2N\mbox{arsh}\dfrac{\sqrt{-\gamma+\lambda}}{2N}. $$
Такой же результат можно получить для всех уравнений кроме последнего. При подстановке замены \eqref{euler_replace} в последнее уравнение, для $ j=N $
\begin{eqnarray*}  
(\lambda-\gamma)  \ch \dfrac{\delta}{N} \left( N-\dfrac{1}{2} \right) = N^2 \left( \ch \dfrac{\delta}{N} \left( N-\dfrac{3}{2} \right) - \ch \dfrac{\delta}{N} \left( N-\dfrac{1}{2} \right) \right) + \\ + \alpha \sum_{k=1}^{n} \:\ch \dfrac{\delta}{N} \left( k-\dfrac{1}{2} \right),
\end{eqnarray*}
можно получить выражение для параметра $ \alpha $:
\begin{equation}\label{alpha_cr}
    \alpha = 2N\sqrt{-\gamma+\lambda}\,\sh\,\dfrac{\delta}{2 N},
\end{equation}
где $ x_k = \dfrac{1}{N} \left( k - \dfrac{1}{2} \right) $.

Для изучения потери устойчивости нулевого решения определим при каких критических значениях параметра $ \alpha $ нулевое собственное значение переходит через мнимую ось. Подстановка $ \lambda=0 $ в выражение \eqref{alpha_cr} приводит к зависимости
\begin{equation}\label{alpha_u}
    \alpha_u = 2N\sqrt{-\gamma}\,\sh\,\dfrac{\delta}{2 N},
\end{equation}
где $ \delta_u = 2N\,\mbox{arsh}\,\dfrac{\sqrt{-\gamma}}{2N} $. Исходя из полученной формулы, справедлива следующая лемма.
\begin{lm}
Для линеаризованной в нуле системы дифференциальных уравнений \eqref{original_system}, \eqref{original_conditions} нулевое состояние равновесия теряет свою устойчивость для значений, параметров $\alpha$ и $\gamma$, связанных формулой \eqref{alpha_u}.
\end{lm}

Для предельного случая $N \rightarrow \infty$, линеаризованную систему \eqref{linearized_system}, \eqref{linearized_conditions} можно свести к следующей краевой задаче:
\begin{equation}\label{linear_system1}
	\dot u = u'' + \gamma u,	
\end{equation}
\begin{equation}\label{linear_system2}	
	u'(0, t) \, = 0, \qquad u'(1, t) \, = \alpha\int\limits_{0}^{1} u(t, y) dy.
\end{equation}
Выполняя для нее также замену \eqref{euler_replace}, с учетом предельного равенства
$$ \lim_{N\to \infty} \delta = \sqrt{-\gamma + \lambda} $$
выражение \eqref{alpha_cr} перепишется в виде
$$ \alpha = -\gamma + \lambda, $$
а подстановка в него $\lambda = 0$ позволяет получить следующую критическую зависимость:
\begin{equation}\label{alpha_div_lim}
    \alpha_u = -\gamma.
\end{equation}

В связи с более простым видом \eqref{alpha_div_lim} функции $\alpha_u$ для предельного случая, подсчет соответствующих критических значений осуществлялся для линеаризованной краевой задачи \eqref{linear_system1}, \eqref{linear_system2}. 

Для системы \eqref{linearized_system}, \eqref{linearized_conditions} изучим динамику поведения критической зависимости $\alpha_u(\gamma)$ для различных значений индекса $k$, где $k$ -- номер узла, с которым дополнительно связано последнее уравнение. В нашем случае количество уравнений $ N $ считалось равным 50. Отметим, что увеличение $N$ слабо сказывается на поведении функций $\alpha_u(\gamma)$ и $\alpha_c(\gamma)$.

Как видно из рис. 1, построенная критическая зависимость $\alpha_u(\gamma)$ позволяет получить области значений параметров, определяющих устойчивость нулевого решения. Так область $ S $ соответствует случаю устойчивого нулевого решения, $ U $ --случаю появления двух состояний равновесия, расположенных симметрично относительно нулевого решения. Отметим, что все результаты локальны и получены в некоторой окрестности нулевого состояния равновесия и достаточно малой окрестности критических кривых.

\section{Локальный анализ системы в окрестности нулевого решения}

Методами малых возмущений (см. \cite{Glyzin}, \cite{Guckenheimer}) построим режим, ответвляющийся от нулевого состояния равновесия системы дифференциальных уравнений \eqref{original_system}, \eqref{original_conditions} для нулевого собственного значения $\lambda$ матрицы соответствующей линеаризованной системы. Для этого введем в рассмотрение малый параметр $\varepsilon$, который косвенно характеризует переход из области $ S $ в область $ U $ по параметру $\alpha$ для фиксированного значения $\gamma$. Здесь $\varepsilon$ принимает вид
\begin{equation}\label{eps_alpha_u}
\varepsilon = \alpha - \alpha_u,
\end{equation}
где $\alpha_u$ вычисляется по формуле \eqref{alpha_u}.

Воспользуемся нормальной формой, которая получается в результате разложения нулевого решения системы \eqref{original_system}, \eqref{original_conditions} по степеням малого параметра
\begin{equation}\label{normal_form}
	u_j = \sqrt{\varepsilon}u_{j,0} + \varepsilon u_{j,1} + \varepsilon^{\frac{3}{2}} u_{j,2} + O(\varepsilon^2), \qquad j = \overline{1, N}.
\end{equation}
Функции $u_j=u_j(s)$ зависят от медленного времени $s=\varepsilon t$, а $u_{j,0}=\rho(s)\ch \delta_u x_j $, где $ \delta_u = 2N\,\mbox{arsh}\,\dfrac{\sqrt{-\gamma}}{2N},\, x_j = -\dfrac{1}{2N} + \dfrac{j}{N} $. 

Подстановка в цепочку дифференциальных уравнений \eqref{original_system}, \eqref{original_conditions} разложения \eqref{normal_form} с учетом \eqref{eps_alpha_u} приводит к последовательно разрешимым системам для векторов $u_{j,0},\, u_{j,1}$ и $u_{j,2}$:
$$ \dot u_{j,0} = N^2(u_{j+1,0} - 2u_{j,0} + u_{j-1,0}) + \gamma u_{j,0}, \quad j = \overline{1, N}, $$
$$ u_{0,0} = u_{1,0}, \quad u_{N+1,0} = u_{N,0} + \dfrac{\alpha_u}{N}u_{k,0}, \qquad 1 \le k < N, $$
\begin{equation}\label{u1_system_u}
	\dot u_{j,1} = N^2(u_{j+1,1} - 2u_{j,1} + u_{j-1,1}) + \gamma u_{j,1},
\end{equation}
\begin{equation}\label{u1_conditions_u}
	u_{0,0} = u_{1,1}, \quad u_{N+1,1} = u_{N,1} + \dfrac{\alpha_u}{N}u_{k,1},
\end{equation}
\begin{equation}\label{u2_system_u}
	\dot u_{j,2} + \frac{\partial u_{j,0}}{\partial s} = N^2(u_{j+1,2} - 2u_{j,2} + u_{j-1,2}) + \gamma u_{j,2} - u_{j,0}^3,
\end{equation}
\begin{equation}\label{u2_conditions_u}
	u_{0,2} = u_{1,2}, \quad u_{N+1,2} = u_{N,2} + \dfrac{\alpha_u}{N}u_{k,2} + u_{k,0}.
\end{equation}
Учитывая, что уравнение \eqref{original_system} содержит кубическое вычитаемое, для функций $u_{j,1}$ система \eqref{u1_system_u}, \eqref{u1_conditions_u} получается однородной. Примем их значения нулевыми. 

Из условий разрешимости системы \eqref{u2_system_u}, \eqref{u2_conditions_u}, можно получить укороченное уравнение на величину $\rho$
\begin{equation}\label{rho_equation}
	\rho' = \phi_0 \rho + d_0 \rho^3,
\end{equation}
где $\phi_0=1$, а коэффициент $ d_0 $ вычисляется по формуле:
\begin{equation}\label{d0_u}
d_0 = \dfrac{5 \delta_u^2 \sh 3\delta_u}{48 \sh \delta_u} - \dfrac{3}{4}.
\end{equation}

Для предельного случая $N \rightarrow \infty$ подстановка в краевую задачу \eqref{system5}, \eqref{system6} нормальной формы вида
\begin{equation}\label{normal_form_lim}
	u = \sqrt{\varepsilon}u_0(x, s) + \varepsilon u_1(x, s) + \varepsilon^{\frac{3}{2}} u_2(x, s) + O(\varepsilon^2),
\end{equation}
где функция $u=u(x, s)$ зависит от медленного времени $s=\varepsilon t$, приводит к цепочке последовательно разрешимых краевых задач для функций $u_0$, $u_1$ и $u_2$
$$ \dot u_0 = u_0'' + \gamma u_0, $$
$$ u_0'(0, s) = 0, \qquad u_0'(1, s) = \alpha_u u_0(x_0, s), $$
\begin{equation}\label{u1_oscillators1_lim}
	\dot u_1 = u_1'' + \gamma u_1,
\end{equation}
\begin{equation}\label{u1_oscillators2_lim}
	u_1'(0, s) = 0, \qquad u_1'(1, s) = \alpha_u u_1(x_0, s),
\end{equation}
\begin{equation}\label{u2_oscillators1_lim}
	\dot u_2 = u_2'' + \gamma u_2 - u_0^3,
\end{equation}
\begin{equation}\label{u2_oscillators2_lim}
	u_2'(0, s) = 0, \qquad u_2'(1, s) = \alpha_u u_2(x_0, s) + u_0(x_0, s).
\end{equation}
Здесь функция $u_0$ будет выглядеть следующим образом
\begin{equation}\label{u0_div_lim}
	u_0 = \rho(s) \ch \sqrt{\gamma} x,
\end{equation}
а функцию $u_1$ можно считать нулевой, исходя из однородности краевой задачи \eqref{u1_oscillators1_lim}, \eqref{u1_oscillators2_lim}. Из условий разрешимости системы \eqref{u2_oscillators1_lim}, \eqref{u2_oscillators2_lim}, можно также получить укороченное уравнение \eqref{rho_equation} на величину $\rho$, для которого $\phi_0=1$, а коэффициент $d_0$ рассчитывается по формуле:
$$ d_0 = -\dfrac{5 \gamma \sh 3\sqrt{-\gamma}}{48 \sh \sqrt{-\gamma}} - \dfrac{3}{4}. $$

Согласно численным результатам, для значений параметра $ \gamma \geqslant 0 $ удавалось выделить и промежутки, где значение $ d_0 $ оказывалось отрицательным, и где эта величина была положительной. Это говорит о том, что для $ \gamma \geqslant 0 $ имело место быть как дивергентная потеря устойчивости, в случае $ d_0 < 0 $, так и грубая, в случае $ d_0 > 0 $. В том случае, если значение параметра $\gamma $ было отрицательным, величина $d_0$ всегда оказывалась положительной, т.е. происходила только грубая потеря устойчивости нулевого решения системы дифференциальных уравнений \eqref{original_system}, \eqref{original_conditions}. График функции $d_0(\gamma)$ показан на рис. 2.

При условии, что $\phi_0>0$ и $d_0<0$, происходит бифуркации типа <<вилка>>. В этом случае уравнение \eqref{rho_equation} имеет ненулевое состояние равновесия $\phi_*=\sqrt{-\dfrac{1}{d_0}}$, причем $\rho$ стремится к этому состоянию равновесия при $s \rightarrow +\infty$. Подставляя в нормальную форму \eqref{normal_form} полученное значение $\rho$, получаем асимптотическое приближение для двух пространственно неоднородных устойчивых состояний равновесия исходной системы \eqref{original_system}, \eqref{original_conditions} (см. также \cite{Glyzin}, \cite{Mardsen}):
\begin{equation}\label{u_j_div}
u_j = \pm \sqrt{- \frac{\varepsilon}{d_0}} \ch \delta_u x_j + O(\varepsilon).
\end{equation}

При условии, что $\phi_0>0$ и $d_0>0$ происходит обратная бифуркация типа <<вилка>>. В этом случае уравнение \eqref{rho_equation} имеет ненулевое состояние равновесия $\rho=\rho_*$, где $\phi_*=\sqrt{\dfrac{1}{d_0}}$. При подстановке полученного значения $\rho$ в нормальную форму \eqref{normal_form}, получаем асимптотическое приближение \eqref{u_j_div} для двух пространственно неоднородных неустойчивых состояний равновесия, стягивающиеся к нулевому решению системы \eqref{original_system}, \eqref{original_conditions} при $\varepsilon \rightarrow 0$ и отбирающие у него устойчивость (см. также \cite{Kuznetsov}, \cite{Hassard}).

\section*{Выводы}

Для краевой задачи параболического типа с дополнительной интегральной внутренней связью во втором краевом условии с помощью перехода к соответствующей ей системе дифференциальных уравнений с диффузионным взаимодействием между соседними элементами была найдена критическая зависимость параметров $\alpha$ и $\gamma$, при которых происходят различные бифуркации нулевого состояния равновесия. Эта функция является важнейшим элементом для построения областей значений параметров, определяющих устойчивость нулевого решения системы дифференциальных уравнений. При значениях параметра $ \alpha $, близких к критическим, была построена нормальная форма и на ее основе выяснены условия появления неоднородных состояний равновесия. При значениях параметра $ \alpha $ близких к критическим, были построены асимптотические формулы для режимов, ответвляющихся от нулевого решения.

Полученные результаты могут быть использованы при решении задач численного моделирования некоторых биофизических процессов (см. например \cite{Gourley}, \cite{Britton}). Вызывает также интерес распространение этих результатов и на другие задачи с дополнительной внутренней связью (см. например \cite{Kashchenko}).
