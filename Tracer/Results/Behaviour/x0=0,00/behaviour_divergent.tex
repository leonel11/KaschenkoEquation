Рассмотрим систему дифференциальных уравнений с диффузионным взаимодействием между соседними элементами и дополнительной внутренней связью

\begin{equation}\label{system1}
\dot{u}_j =  N^2(u_{j+1} - 2u_j + u_{j-1}) + \gamma u_j - u_j^3, \qquad j = \overline{1, N}, 
\end{equation}
\begin{equation}\label{system2}
u_0 = u_1, \; u_{N+1} = u_N + \dfrac{\alpha}{N}u_k, \qquad 1 \leqslant k < N.
\end{equation}
где $ u_j = u_j(t) $ --- гладкие функции при $ t \geqslant 0 $, величина $ N \gg 1 $, а параметры $ \alpha, \beta, \gamma $ --- действительные числа. Индекс $ k \in \mathbb{N} $ позволяет определить дополнительную внутреннюю связь между элементами $ u_N $ и $ u_k $. 

Изучим поведение нулевого состояния равновесия системы \eqref{system1}, \eqref{system2} вблизи критических значений $\alpha=\alpha_u$ для различных значений $\gamma$ при $k=1$. Величина $\alpha_u$ рассчитывается по формуле
\begin{equation}\label{alpha_div}
    \alpha_u = \frac{ \sqrt{-\gamma} \, \sh \delta_u }{ \ch\delta_u x_k },
\end{equation}
где $ \delta_u = 2N\,\mbox{arsh}\,\dfrac{\sqrt{-\gamma}}{2N} $, а $ x_k = \dfrac{1}{N} \left( k - \dfrac{1}{2} \right) $. Схематическая визуализация критических значений $\alpha$ и $\gamma$, а также график функции $\alpha_u(\gamma)$ показаны на рис. 1. Кривая $ \alpha_u $, соответствующая случаю дивергентной потери устойчивости нулевого состояния равновесия, показана синим цветом, а кривая $ \alpha_c $, соответствующая случаю колебательной потери устойчивости нулевого решения, показана красным цветом. Кривые $\alpha_u$ и $\alpha_c$ пересекаются в точке $B$ с координатами $(\gamma_*, \alpha_*)$, где $ \gamma_* > 0 $ и $ \alpha_* < 0 $. Так, в случае $k=1 \; (x_0 = 0)$ значение $\gamma_* \approx 4.116$, а $\alpha_* \approx -1.82$.

\noindent Кривые $\alpha_u$ и $\alpha_c$ в данном случае являются важнейшими элементами построения областей значений параметров $(\alpha, \gamma)$, определяющих устойчивость нулевого состояния равновесия системы \eqref{system1}, \eqref{system2}. Так область $S$ соответствует случаю устойчивого нулевого решения, $U$ -- случаю появления двух симметричных состояний равновесия, а в области $C$ наблюдается наличие цикла вблизи неустойчивого нулевого решения. Отметим, что все результаты локальны и получены в некоторой окрестности нулевого состояния равновесия и достаточно малой окрестности критических кривых.

С помощью предельного перехода $N \rightarrow \infty$, систему дифференциальных уравнений \eqref{system1}, \eqref{system2} можно свести к непрерывной краевой задаче

\begin{equation}\label{problem1}
\dot{u} = u'' + \gamma u - u^3,
\end{equation}
\begin{equation}\label{problem2}
u'(0, t) \, = 0, \qquad u'(1, t) \, = \alpha\,u(x_0, t).
\end{equation}
Здесь функция $ u(x, t)$ является гладкой при $t\ge 0$ и $x\in[0,1]$, а величина $ x_0 \in [0, 1) $ вместо индекса $k$, определяет отклонение во втором краевом условии.  В нашем случае, поскольку величина $k=1$, значение $x_0 = 0$.

С помощью вспомогательной функции $v(x)$ сведем задачу \eqref{problem1}, \eqref{problem2} к следующей системе:

\begin{equation}\label{vsystem1}
 \begin{cases}
   u' = v
   \\
   v' = -\gamma u + u^3.
 \end{cases},
\end{equation}
Для вектора $ (u, v)^T $ определим следующие условие на его компоненты
\begin{equation}\label{vsystem2}
u(0) \, = a, \quad v(0) \, = 0,
\end{equation}
где $ a $ --- действительное число.

\begin{thm}
Введем в рассмотрение нечетную скалярную функцию $ F $, определяемую по формуле
\begin{equation} \label{fa} 
F(a) = v(1) - \alpha\,u(x_0).
\end{equation}
Пусть $ a_*\in [0, A]$ такое, что $ F(a_*) = 0 $. Тогда функция $ u_*(x) $, удовлетворяющая условиям
\begin{equation} \label{vsystem2star} 
u_*(0) \, = a_*, \qquad u_*'(0) \, = 0
\end{equation}
будет являться решением системы \eqref{vsystem1} и краевой задачи \eqref{problem1}, \eqref{problem2} соответственно.
\end{thm}

Определим условия дивергентной потери устойчивости нулевого решения краевой задачи \eqref{problem1}, \eqref{problem2}, а также исследуем фазовые перестройки, проходящие в его малой окрестности нулевого состояния равновесия. Для этого с помощью функций $ u(x) $ и $ v(x) $, построенных методом Рунге-Кутты, и различных значений $ a \in [0, A]$ найдем корни $a_*$ уравнения
\begin{equation}\label{f_equation}
F(a) = 0,
\end{equation} 
которые соответствуют состояниям равновесия системы краевой задачи \eqref{problem1}, \eqref{problem2}.

Локальный анализ поведения системы \eqref{system1}, \eqref{system2} в окрестности нулевого решения для $k=1 \; (x_0=0)$ и $\widetilde{\gamma} \approx 4.039$ позволил численно доказать следующие теоремы.

\begin{thm}
Пусть $\varepsilon=\alpha-\alpha_u$. Тогда для любого $\gamma < \widetilde{\gamma}$ существует $\varepsilon_0$ такое, что для $\varepsilon \in (0,\varepsilon_0]$ система \eqref{system1}, \eqref{system2} имеет в окрестности нуля два пространственно неоднородных устойчивых режима.
\end{thm}
\begin{thm}
Пусть $\varepsilon=\alpha-\alpha_u$. Тогда для любого $ \widetilde{\gamma} < \gamma < \gamma_* $ существует $\varepsilon_0$ такое, что для $\varepsilon \in (0,\varepsilon_0]$ система \eqref{system1}, \eqref{system2} имеет в окрестности нуля два пространственно неоднородных неустойчивых режима.
\end{thm}

При изменении параметра $\alpha$ вблизи критических значений $\alpha_u$, для любого $\gamma$ такого, что $\gamma < \widetilde{\gamma}$ наблюдается один и тот же сценарий поведения функции $F(a)$. Для определенности возьмем $ \gamma = -2.0 $ и начнем менять параметр $\alpha$ вблизи критического значения $\alpha_u$. Численный анализ поведения функции $F(a)$ позволяет получить следующую последовательность фазовых перестроек для нулевого состояния равновесия для системы \eqref{problem1}, \eqref{problem2}:

\begin{enumerate}
\item При $\alpha < \alpha_u$, $\alpha_u \approx 2.737$, где  уравнение \eqref{f_equation} имеет единственный нулевой корень. Другими словами, нулевое состояние равновесия краевой задачи \eqref{problem1}, \eqref{problem2} будет устойчивым. График функции $F(a)$ для данного случая при $a>0$ изображен на рис. 2.

Это означает, что от нулевого состояния равновесия краевой задачи \eqref{problem1}, \eqref{problem2}, ответвляется пара устойчивых симметричных состояний равновесия, в результате чего нуль теряет свою устойчивость. Полученный сценарий поведения функции $F(a)$ верен для любых $\gamma < \widetilde{\gamma}$ и $\alpha > \alpha_c$. Отличия от рассмотренного случая состоят лишь в числовых значениях параметра $\alpha_u$.

При изменении параметра $\alpha$ вблизи критических значений $\alpha_u$, для любого $\gamma$ такого, что $\widetilde{\gamma} < \gamma < \gamma_*$ наблюдается один и тот же сценарий поведения функции $F(a)$. Для определенности возьмем $ \gamma = -4.1 $ и начнем менять параметр $\alpha$ вблизи критического значения $\alpha_u$. Численный анализ поведения функции $F(a)$ позволяет получить следующую последовательность фазовых перестроек для нулевого состояния равновесия для системы \eqref{problem1}, \eqref{problem2}:

\begin{enumerate}
\item При $\alpha<\alpha_1$, где $\alpha_1 \approx -1.823$, уравнение \eqref{f_equation} имеет единственный нулевой корень. Полученный сценарий поведения функции $F(a)$ верен для любых $\widetilde{\gamma} < \gamma < \gamma_*$ и $\alpha > \alpha_c$. Отличия от рассмотренного случая состоят лишь в числовых значениях параметра $\alpha_u$. Начиная со значения $\gamma > \gamma_*$ фазовые перестройки будут происходить с неустойчивым нулевым состоянием равновесия краевой задачи \eqref{problem1}, \eqref{problem2}.

Вместе с приведенным выше анализом фазовых перестроек, численно можно доказать следующие теоремы.

\begin{thm}
Пусть $\gamma < \widetilde{\gamma}$. Тогда на плоскости параметров $(\alpha, \gamma)$, при переходе через кривую $\alpha_u(\gamma)$ от нулевого решения краевой задачи \eqref{problem1}, \eqref{problem2} ответвляются два устойчивых симметричных состояния равновесия.
\end{thm}
\begin{thm}
Пусть $\widetilde{\gamma} < \gamma < \gamma_*$. Тогда на плоскости параметров $(\alpha, \gamma)$, при переходе через кривую $\alpha_u(\gamma)$ нулевое решение краевой задачи \eqref{problem1}, \eqref{problem2} грубо теряет свою устойчивость: к нулевому состоянию равновесия равномерно подходит пара неустойчивых состояний равновесия и отбирает у него устойчивость.
\end{thm}
